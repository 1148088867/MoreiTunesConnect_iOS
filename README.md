# DoubleiTunes
伪iTunes Connect多开

- 外包人员必备
- 多账号管理
- 本地信息AES加密
- 粘贴板获取信息，自动录入
- 自动提交表单
- 开源
- DIY

    {账号/密码>账号/密码>账号/密码}
    或
    {账号/密码/备注>账号/密码>账号/密码/备注}
